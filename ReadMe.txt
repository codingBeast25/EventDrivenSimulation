Program to run on linux environment command line.
1)Open the directory where the assignment is,

2)Then type 'make' to compile all the files of the assignment.

3)After compiling the files,

	i. To run the assignment,type 'A2' followed by the precise name of the input file with .txt extension
	   For example, A2 <fileName.txt> (A2 nameOfInputFile.txt)

	ii. To run the tests for the assignment, type 'runTest' to run the tests.
